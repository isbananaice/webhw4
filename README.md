# GPU Price Orbit

A simple local GPU price tracker with an Express.js API, SQLite storage, and a static frontend.

## Run locally

1. Install dependencies:
   - `npm install`
2. Start the server:
   - `npm start`
3. Open the site in the browser:
   - `http://localhost:3000`

## Features

- Add price entries with date, product name, and price.
- Persist data in SQLite so entries survive refreshes.
- Filter the price history by product name.
